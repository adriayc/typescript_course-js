STORE APP

* Initialize the app
  - Create React + TS App (VITE)
    $ npm create vite@latest store-app -- --template react-ts
    $ cd store-app
    $ code .              // Open in VSCode
    $ npm install
    $ npm run dev
    $ npm install && npm run dev
  - Install dependencies
    + Redux Toolkit
      $ npm install @reduxjs/toolkit
      $ npm install react-redux
    + React Router DOM
      $ npm install react-router-dom
    + Axios
      $ npm install axios
