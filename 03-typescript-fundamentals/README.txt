TYPESCRIPT FUNDAMENTALS

* Initialize the app
  - Create vanilla-ts App (VITE)
    $ npm create vite@latest typescript-fundamentals -- --template vanilla-ts
    $ cd typescript-fundamentals
    $ npm install
    $ npm run dev
  - Show error when generation the buil production
    $ npm run build
  - Install dependencies
    + Zod
      $ npm install zod
    + Bcryptjs
      $ npm i bcryptjs
      $ npm install --save @types/bcryptjs
