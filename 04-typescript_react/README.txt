TYPESCRIPT AND REACT

* Initialize the app
  - Create react-ts App (VITE)
    $ npm create vite@latest typescript-react -- --template react-ts
    $ cd typescript-react
    $ npm install
    $ npm run dev
  - Install dependencies
    + Zod
      $ npm install zod
    + Axios
      $ npm install axios
    + React Query
      $ npm i @tanstack/react-query
    + Redux Toolkit
      $ npm install @reduxjs/toolkit
      $ npm install react-redux
